비밀번호 변경

새 비밀번호를 입력하세요. (최소 6자)

사용 안내

전사유니폼 주문서를 넣으면 선수별 출력 파일(PDF/EPS)을 자동으로 만들어 ZIP으로 받는 사내 도구입니다.

1. 첫 로그인 후 비밀번호 변경 (꼭!)

  1. 관리자에게 받은 이메일과 임시 비밀번호로 로그인합니다.
  2. 로그인하면 우측 상단의 내 이메일이 보입니다 → 클릭 → [비밀번호 변경] 선택.
  3. 새 비밀번호를 2번 입력(6자 이상, 두 칸 일치) → 변경. 재로그인은 필요 없습니다.

2. 출력 파일 만들기 — 5단계

화면 위쪽 순서대로 진행하고, 오른쪽 주문 요약에서 진행 상태를 확인합니다.

  1. 패턴 선택 — 만들 유니폼 패턴 카드를 클릭합니다.
  2. 디자인 — 번호·이름 없는 빈 본체(PDF 호환 AI/PDF)를 업로드합니다. 업로드하면 자동 점검이 돌고, 통과해야 다음 단계가 켜집니다.
  3. 주문서 — STIZ 주문서 엑셀(.xlsx)을 업로드해 이름·배번·사이즈·수량을 확인합니다.
  4. 생성 — 출력 형식(PDF/EPS) 확인 후 생성합니다. 큰 주문은 시간이 걸립니다.
  5. 검수 & 다운로드 — 선수별 미리보기·검증 결과 확인 후 ZIP 다운로드.

3. 로그아웃

우측 상단 내 이메일 클릭 → [로그아웃]. 공용 PC에서는 꼭 로그아웃하세요.

4. 자주 묻는 것 (FAQ)

  • 첫 화면이 느려요 — 서버를 깨우는 중입니다(약 30초). 잠시 기다리면 떠요.
  • "로그인이 필요합니다" — 세션이 만료된 것. 다시 로그인하세요.
  • 디자인 "본체 누락" 경고 — 일러스트레이터 저장 시 "PDF 호환 파일 만들기" 체크, 그리고 완성본 말고 번호·이름 없는 빈 본체를 업로드하세요.
  • 패턴 등록·설정이 안 돼요 — 관리자(admin) 권한 계정만 가능합니다.
  • 비밀번호를 잊었어요 — 관리자에게 임시 비밀번호 재발급을 요청하세요.

문제가 계속되면 화면을 캡처해 관리자에게 전달해 주세요.
