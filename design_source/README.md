# design_source — 디자인 원본 .ai (gitignore 대상)

이 폴더의 `*.ai` 파일은 용량이 커서(1.7~1.9MB) git에 포함하지 않습니다(.gitignore).
**clone 시 디자이너에게 아래 파일을 별도로 받아 이 폴더에 넣어야** job/grade가 동작합니다.

## 필요한 파일
| 파일명 | 용도 | 참조 위치 |
|--------|------|----------|
| `연세대_V넥_빈템플릿_본체포함_XL.ai` | **디자인 본체(base)** — 파란 본체+밴드+빨간 재단선, 번호·이름 없는 깨끗한 빈 템플릿(PDF콘텐츠 ≈88KB) | `data/patterns/농구_V넥_양면/preset.json`의 `design_file` |
| `연세대_V넥_숫자글리프셋_아트보드2.ai` | **번호 글리프셋 소스** — 0~9 athletic 블록체 outline. `number-glyphs` 명령으로 `number_glyphs.json` 추출(이미 추출본 커밋됨) | `data/patterns/농구_V넥_양면/number_glyphs.json` |
| `연세대_V넥_완성본_본체포함_XL.ai` | 참고용 완성본(번호·이름 포함 기준) | (검수 비교용) |

## ⚠️ 디자이너 운영 규칙 (중요)
- 본체 .ai는 일러스트레이터에서 **"PDF 호환 파일 만들기(Create PDF Compatible File)" 옵션을 켜고 저장**해야 합니다.
  - 이 옵션이 꺼지면 파란 본체·재단선이 일러스트 전용 데이터에만 남아 PDF 콘텐츠가 ~3.4KB로 떨어지고, 합성 출력에 본체가 안 나옵니다(흰 화면).
- 빈 템플릿은 번호·이름 텍스트를 모두 제거한 상태여야 합니다(겹침 방지).
