# 유니폼 그레이딩 도구 — 직원 실행 가이드

> 이 한 장만 보면 됩니다. 어려운 용어 없이 따라 하기만 하세요.

---

## 1. 켜기 (시동)

1. 프로그램 폴더에서 **`run.bat`** 파일을 **더블클릭**합니다.
2. 검은 창이 뜨고 "서버를 켜는 중입니다…" 메시지가 나옵니다.
3. 잠시 후 **브라우저(크롬/엣지)가 자동으로 열리고** 작업 화면이 나타납니다.
   - 자동으로 안 열리면 브라우저 주소창에 **`localhost:8000`** 을 직접 입력하세요.

> 💡 **검은 창은 닫지 마세요.** 그 창이 "엔진이 돌아가는 곳"입니다.
> 작업이 다 끝나면 그 검은 창을 닫으면 프로그램도 함께 꺼집니다.

---

## 2. 사용하기 (브라우저 화면)

화면 위쪽 단계 표시(스테퍼)를 따라 **왼쪽 → 오른쪽** 순서로 진행합니다.

| 단계 | 무엇을 하나요 |
|------|--------------|
| ① 패턴 선택 | 만들 유니폼 종류(예: **농구 V넥**)를 카드에서 고릅니다. |
| ② 디자인 업로드 | **빈 본체 도안(.ai/.pdf)** 을 끌어다 놓습니다. 자동으로 점검합니다. |
| ③ 주문서 업로드 | 주문 엑셀(**.xlsx**)을 올리면 선수 명단 표가 자동으로 채워집니다. |
| ④ 생성 | 출력 형식(PDF / EPS / 둘 다)을 고르고 **생성**을 누릅니다. 진행바가 채워집니다. |
| ⑤ 검수 | 선수별 미리보기와 검증 배지를 확인합니다. 문제없으면 **ZIP 다운로드**. |

### 디자인 업로드 점검 안내 (②단계)
업로드하면 자동으로 신호등처럼 표시됩니다.
- 🟢 **정상(pass)**: 빈 본체 + 투명도 없음 → 다음 단계로 진행하세요.
- 🟡 **경고(warn)**: "빈 본체를 올려주세요" → 완성본(번호·이름이 이미 박힌 도안)을 올린 경우입니다. 빈 본체로 교체 권장.
- 🔴 **차단(fail)**: "본체가 PDF에 안 들어감" 등 → 일러스트레이터에서 **PDF 호환 저장**으로 다시 내보낸 .ai 를 올려주세요.

### 검수에서 "건너뜀"이 보일 때 (⑤단계)
- 일부 사이즈(예: **3XL**)는 원본 도안 문제로 자동 제외될 수 있습니다.
- "건너뜀 N건" 배너에 사유가 표시됩니다. 해당 사이즈는 별도 처리하세요.

## 3. 다른 PC에 설치하기 (배포)

이 도구는 설치 프로그램이 따로 없습니다. **폴더를 통째로 복사**하면 됩니다.

1. 이 프로그램 폴더 전체(`grader-v2`)를 USB나 사내망으로 새 PC에 **복사**합니다.
2. 새 PC에 **Python 3.11** 이 없으면 먼저 설치합니다.
   - https://www.python.org/downloads/ → 설치 시 **"Add Python to PATH" 체크박스를 꼭 켜기**.
3. 새 PC에서 `run.bat` 을 한 번 더블클릭합니다.
   - 처음이면 "라이브러리를 설치하라"는 안내가 나옵니다. 검은 창에 안내된 명령
     (`python -m pip install -r requirements.txt`)을 복사해 붙여넣고 실행하세요. (인터넷 필요)
4. 설치가 끝나면 다시 `run.bat` 을 더블클릭하면 정상 실행됩니다.

## 4. EPS 출력을 쓰려면 (Ghostscript)

- **PDF만 출력**한다면 아무것도 설치할 필요 없습니다. 바로 사용하세요.
- **EPS 형식**(인쇄소 입고용)이 필요하면 **Ghostscript** 라는 무료 프로그램이 추가로 필요합니다.

### 설치 방법
1. https://ghostscript.com/releases/gsdnld.html 에서 **Ghostscript (64-bit) for Windows** 를 받아 설치합니다.
2. 보통 `C:\Program Files\gs\gs10.04.0\bin\gswin64c.exe` 에 설치됩니다.
3. 도구의 **설정 화면**에서 이 경로를 입력해 두면 EPS 변환이 자동으로 됩니다.
   (경로를 비워 둬도 도구가 알아서 표준 위치를 찾아봅니다.)

> 💡 Ghostscript가 없거나 경로가 틀려도 **PDF는 정상 생성**됩니다.
> 이때 EPS만 "건너뜀"으로 표시되고 프로그램은 멈추지 않습니다.

## 5. 문제가 생기면 (자주 묻는 질문)

| 증상 | 원인 / 해결 |
|------|------------|
| 더블클릭해도 검은 창만 깜빡이고 꺼짐 | 파이썬 미설치 → 위 **3-2** 대로 Python 3.11 설치 |
| "라이브러리가 설치되지 않았습니다" | 검은 창의 안내 명령을 복사해 실행 (인터넷 필요) |
| 브라우저가 자동으로 안 열림 | 주소창에 `localhost:8000` 직접 입력 |
| 화면이 안 뜨고 "연결할 수 없음" | 검은 창이 켜져 있는지 확인. 꺼졌으면 `run.bat` 다시 실행 |
| 사이드바에 "서버 연결 안 됨" 빨간 표시 | 검은 창(서버)이 꺼진 것. `run.bat` 다시 실행 |
| EPS가 "건너뜀"으로만 나옴 | Ghostscript 미설치/경로 오류 → 위 **4** 참고 (PDF는 정상) |

> 그래도 안 되면 검은 창에 나온 빨간 글씨(오류 메시지)를 캡처해서 담당자에게 보내주세요.
