# data/fonts — 배번·이름 렌더용 폰트

A-4(배번·이름 자동 입력)는 글자를 **벡터 경로로 아웃라인화**해 출력에 그린다(폰트 임베드 아님).
그러려면 이 폴더에 **ttf/otf 폰트 파일**이 있어야 한다. 폰트가 없으면 "폰트가 없어서 안 된다"는 오류가 난다.

## 지금 들어있는 것 (운영용 — Pretendard 설치 완료)
- `Pretendard-Black.otf` — **배번(숫자)용**. 무겁고 또렷. (preset `number_area.font`)
- `Pretendard-Bold.otf` — **이름(한글)용**. (preset `name_area.font`)
  - 둘 다 **한글·숫자·영문 전부 커버 확인됨**. OFL(무료·상업 인쇄 가능).
- `_TEST_LiberationSans-Bold.ttf` — (무시/삭제 가능) 초기 숫자 테스트용 라틴 폰트. 한글 불가.

> preset.json(`data/patterns/농구_U넥_양면/preset.json`)의 `number_area`/`name_area`에 위 폰트 경로가
> 이미 연결돼 있다(`color_cmyk:[0,0,0,0]`=흰색, 파란 유니폼에 흰 글자 기준). 색·정렬은 디자인에 맞게 조정.

## 폰트를 바꾸거나 추가하려면
1. **한글 + 숫자 지원 폰트**(ttf/otf)를 이 폴더에 넣는다.
2. 배번(숫자)과 이름(한글)을 **다른 폰트**로 쓰려면 2개를 넣고 preset에서 각각 지정.

## 폰트 선택 가이드 (중요: 라이선스)
- 출력물은 **상업 인쇄물**이다. 아웃라인화해도 폰트 디자인을 사용하는 것이므로 **자유 라이선스(OFL 등) 폰트**를 권장.
- **추천(무료·OFL·한글+숫자)**: **Pretendard** — 이 프로젝트의 브랜드 폰트(웹 UI에도 사용 중). 공식 배포처에서 `.ttf`/`.otf`를 받아 이 폴더에 넣으면 됨.
- 배번 숫자는 더 두껍고 각진 '유니폼 넘버' 느낌을 원할 수 있음. 그 경우 해당 폰트의 라이선스(상업 인쇄/아웃라인 허용)를 반드시 확인.
- ❌ **Windows 기본 폰트(맑은 고딕 등)는 피할 것** — 상업 출력 라이선스가 애매하다.
- 기준 디자인에 이미 박힌 번호("20")와 **같은 서체로 맞추려면** 디자이너에게 그 폰트 파일을 받아야 한다.

## CLI(구현)에 전달
- preset.json 또는 job 호출 시 폰트 경로를 이 폴더 기준으로 지정:
  - 예) `number_area.font = "data/fonts/<배번폰트>.ttf"`, `name_area.font = "data/fonts/<이름폰트>.ttf"`
- 글리프 누락(폰트에 없는 글자) 시 해당 선수만 경고로 처리하고 나머지는 진행(부분 실패 허용).
